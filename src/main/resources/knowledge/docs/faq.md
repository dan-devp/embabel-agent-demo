# Quaddel GmbH – Häufig gestellte Fragen (FAQ)

**Wo hat Quaddel seinen Sitz?**

## Produkte

**Kann QuaddelInsight mit unserer bestehenden SAP-Umgebung verbunden werden?**
Ja. QuaddelInsight unterstützt native SAP-Konnektoren für S/4HANA, SAP BW und SAP Analytics Cloud. Setup dauert typisch 2–3 Tage.

**Ist QuaddelFlow auch für kleine Teams geeignet?**
Ja, ab 1 User nutzbar. Es gibt einen kostenlosen 30-Tage-Test ohne Kreditkarte.

**Welche Datenquellen unterstützt QuaddelInsight?**
SQL-Datenbanken (PostgreSQL, MySQL, MSSQL, Oracle), NoSQL (MongoDB, Elasticsearch), REST APIs, Excel/CSV, SAP, Salesforce, HubSpot.

**Läuft QuaddelAI Assistant auf unseren eigenen Servern?**
Ja. On-Premise-Deployment ist der Standardfall für Unternehmenskunden. Keine Daten verlassen Ihre Infrastruktur.

**Gibt es eine Mobile App?**
QuaddelInsight hat eine responsive Web-App für mobile Nutzung. Eine native iOS/Android App ist für Q3 2025 geplant.

## Datenschutz & Sicherheit

**Ist Quaddel DSGVO-konform?**
Ja. Alle Produkte sind DSGVO-konform entwickelt. Wir schließen Auftragsverarbeitungsverträge (AVV) nach Art. 28 DSGVO ab.

**Wo werden Daten gespeichert?**
Bei SaaS-Produkten ausschließlich in deutschen Rechenzentren (Frankfurt a.M., zertifiziert nach ISO 27001).

**Unterstützen Sie Single Sign-On (SSO)?**
Ja. SAML 2.0, OAuth 2.0 und LDAP/Active Directory werden unterstützt.
